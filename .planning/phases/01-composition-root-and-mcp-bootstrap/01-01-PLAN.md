<objective>
Build the single composition root for the llm-wiki application in a new `@llm-wiki/common` workspace package, and extend `WikiConfig` / `ConfigLoader` with the `mcp.host` + `mcp.port` fields needed by the transport layer (plan 01-02).

Purpose: Satisfy WIRE-01 (single composition root instantiates all adapters and injects into services via constructor) and WIRE-02 (ConfigLoader carries the new `mcp.*` fields through shared -> local -> env precedence). This plan produces the `AppServices` contract that plan 01-02 consumes, so the interface is defined in Wave 0 and the transport has no reason to reach into `@llm-wiki/infra` directly.

Output:
- New workspace package `@llm-wiki/common` with `buildContainer(config): AppServices`.
- `WikiConfig.mcp` + `LLM_WIKI_MCP_HOST` / `LLM_WIKI_MCP_PORT` env overrides (port coerced to number).
- Unit tests proving container shape and config precedence.
</objective>

         );

         const searchDbPath = path.resolve(wikiRoot, config.search.db_path);
         const searchEngine = new RuVectorSearchEngine(searchDbPath, embeddingClient);

         const sanitizer = new SanitizationService({
           enabled: config.sanitization.enabled,
           mode: config.sanitization.mode,
           customPatterns: config.sanitization.custom_patterns,
           allowlist: config.sanitization.allowlist,
         });

         const remember = new RememberService(fileStore, verbatimStore, sanitizer);
         const recall = new RecallService(fileStore, verbatimStore, projectResolver);
         const query = new QueryService(searchEngine, llmClient, projectResolver, fileStore);
         const status = new WikiStatusService(fileStore, verbatimStore, searchEngine, stateStore);
         const ingest = new IngestService(
           sourceReader, llmClient, searchEngine, versionControl,
           fileStore, fileStoreFactory, stateStore,
         );

           mainRepoRoot: wikiRoot,
           mainFileStore: fileStore,
           mainVerbatimStore: verbatimStore,
           versionControl,
           searchEngine,
           fileStoreFactory,
           verbatimStoreFactory,
           stateStore,
           archiver,

           makePromotePhase: (fs) => new PromotePhase(fs, llmClient),

         });
         const import_ = new ImportService({
           readers: new Map([['claude-code', new ClaudeCodeMemoryReader()]]),
           verbatimStore, stateStore,
           agentConfigs: {},
         });

       }
